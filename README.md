# VinewoodBowl
## Description
Fame or Shame customized stage for the Vinewood Bowl. Great for roleplaying karaoke events.
## Dependencies
* Should run on any FXServer
## Setup
* Install into resources/[map]
* Add the following line to your server.cfg
```
start VinewoodBowl
```
